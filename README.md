# 🎤 음어그 (U-U-G)

> 스피치 및 대화 습관 개선 도우미

---

## 📌 프로젝트 소개

**음어그(U-U-G)**는
발표, 면접, 일상 대화에서 나타나는 **습관어(“음…”, “어…”, “그…”)와 비언어적 요소**를 분석하여
사용자의 커뮤니케이션 능력 향상을 돕는 서비스입니다.

사용자는 실시간 연습 또는 녹화 파일 분석을 통해
자신의 스피치 패턴을 **객관적으로 파악하고 개선**할 수 있습니다.

---

## 🎯 핵심 목표

- 🎙️ 습관어 및 침묵 구간 감지
- ⚡ 실시간 스피치 피드백 제공
- 📊 분석 리포트를 통한 개선 가이드 제공
- 📈 반복 학습을 통한 커뮤니케이션 능력 향상

---

## 🧱 Tech Stack

### ⚙️ Core

- Bun
  빠른 패키지 관리 및 실행 환경

- Next.js
  App Router 기반의 React 프레임워크

- TypeScript
  정적 타입 기반 안정적인 개발 환경

---

### 🎨 Styling

- Tailwind CSS
  빠르고 일관된 UI 개발을 위한 스타일링

---

### 🔄 Data Fetching

- TanStack Query
  서버 상태 관리 및 캐싱 처리

---

### 🧹 Code Quality

- ESLint
  코드 품질 및 잠재적 오류 방지

- Prettier
  코드 스타일 자동 정렬

---

### 🧪 Testing

- Playwright
  사용자 시나리오 기반 E2E 테스트

---

## 🚀 주요 기능 (MVP)

- 🎤 실시간 스피치 분석 (습관어, 말 속도, 침묵 구간)
- 👀 시선 및 표정 피드백 (비언어적 요소 분석)
- 📁 녹음/녹화 파일 업로드 분석
- 📊 시각화된 분석 리포트 제공
- 🎯 상황별 스피치 연습 (면접 / 발표 / 일상)
- 📈 개인 기록 및 개선 추이 관리

## 🛠️ Getting Started

```bash
bun install
```

## 🧹 Code Convention

```bash
# lint
bun lint

# lint check
bun lint:check

# format
bun format

# type check
bun type-check
```
